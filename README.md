 # SCT_DS_4_US_Accident_Analysis

## Overview
This project performs Exploratory Data Analysis (EDA) on the US Accidents dataset using Python. The goal is to identify accident patterns based on severity, weather conditions, time, location, and road features.

## Dataset
- Dataset: US Accidents (2016–2023)
- Source: Kaggle
- Records analyzed: 100,000 samples
- Features: 46 columns

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Google Colab

## Tasks Performed
### 1. Data Exploration
- Loaded dataset
- Checked data types and missing values
- Viewed sample records

### 2. Accident Severity Analysis
- Analyzed accident severity distribution
- Visualized severity levels using bar charts

### 3. Weather Condition Analysis
- Identified top weather conditions during accidents
- Visualized weather-related accident trends

### 4. Time-Based Analysis
- Extracted accident hours
- Analyzed accident frequency throughout the day

### 5. Geographic Analysis
- Created accident hotspot visualization
- Examined accident locations using latitude and longitude

### 6. State-Wise Analysis
- Identified states with the highest accident counts

### 7. Road Feature Analysis
- Investigated accident occurrences near:
  - Traffic Signals
  - Junctions
  - Crossings
  - Stop Signs

## Key Insights
- Severity levels 2 and 3 account for most accidents.
- Most accidents occurred during clear weather conditions.
- Accident frequency peaks during busy traffic hours.
- California recorded the highest number of accidents in the analyzed sample.
- Traffic signals and junctions are common factors associated with accident locations.
